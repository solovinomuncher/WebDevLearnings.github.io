What is CSS?

Without CSS, HTML file has internal stylesheets with basic default styles to be basically readable. With CSS, HTML file has text styling, layout, and animations.

CSS rules start with selector, then curly braces, then declarations (which are property and value pairs separated by colons and ending with semicolons).

Getting Started with CSS

There are three different ways to apply CSS to an HTML doc:

  • Linking to CSS doc from head of HTML doc (most useful/usual)
  • 2
  • 3

To link CSS doc from head of HTML doc, use link element with rel and href attributes. rel tells the browser that we have a stylesheet (rel="stylesheet") and href tells the browser the location of that stylesheet (href="styles.css").

The selectors target an HTML element to apply the declarations to. There can be multiple selectors. To select specific elements, use the class attribute. To target specific elements, start with period (.) and name of class (e.g. class="special" -> .special {color: blue;}). To target specific subsets of elements, start with element, period, and class name (e.g. li.special {} to target li elements that have a class of special).

It is sometimes best to bypass element and refer to class.
